About

This web application is for Employers to manage their workforce smoothly and digitally. This application gives power to Human Resources department and Backend Admins by allowing them toh manage every information of each employees in every department. Daily attendance system is there which allows Employees and HRs to track attendance of each worker.Personalized dashboards are given to admin users and employees user which allows them to many functionalities.
This web application is developed using Django v-3.2.4 as a project by Reetik Dhaka.
